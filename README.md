# CalculatorProject

## Описание проекта

CalculatorProject — учебный проект по дисциплине «Управление проектами».

Программа представляет собой простой калькулятор с графическим интерфейсом, разработанный на языке C# с использованием Windows Forms.

---

## Возможности программы

- сложение;
- вычитание;
- умножение;
- деление;
- очистка поля;
- отображение выражения;
- защита от деления на ноль.

---

## Используемые технологии

- C#

- Windows Forms
- Visual Studio 2022
- GitHub
